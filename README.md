# 🕵️ Fake News Detector (Frontend Project)

A modern and interactive **Fake News Checker Web App** built using **HTML, CSS, and JavaScript**.  
It analyzes user-input news text and provides a **real or fake prediction with confidence level**.

---

## 🚀 Features

- 🔍 Analyze news headlines or articles instantly  
- 📊 Confidence meter with visual progress bar  
- 🎨 Modern **Blue & Black UI Theme**  
- ⚡ Fast and lightweight (no backend required)  
- 📱 Fully responsive design  
- 🧠 Smart keyword-based detection system  

---

## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling & animations  
- **JavaScript (Vanilla JS)** – Logic & interactivity  

---

## 🧠 How It Works

- The app scans the input text for **predefined keywords**
- Words like *"shocking", "viral", "hoax"* increase fake score  
- Words like *"official", "research", "confirmed"* increase real score  
- Based on scores, it predicts:
  - ✅ Likely Real News  
  - ⚠️ Likely Fake News  
- Displays a **confidence percentage**

---

## ⚠️ Disclaimer

This project uses a **basic keyword-based approach** and is for **educational purposes only**.  
It is **not 100% accurate** and should not be used for real-world fact-checking.
